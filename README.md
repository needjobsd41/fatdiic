在开始订购之前，确保您的 Visa 信用卡已经开通了国际支付功能，并确认信用卡的余额足以支付服务费用。

访问 ChatGPT 官方网站并登录到您的账户。如果您还没有账户，需要先注册一个新的账户。

### 选择服务

在网站的服务页面，选择 ChatGPT Plus 订阅服务。点击“订购”或“升级”按钮，进入支付界面。

### 输入信用卡信息

确认支付信息后，点击“提交”按钮完成支付过程。支付成功后，系统会自动将您的账户升级到 ChatGPT Plus。
